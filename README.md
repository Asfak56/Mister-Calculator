# Voice Calculator
A modern Android calculator built using Kotlin and Jetpack Compose.

## Features
- Voice Input
- Expression & Result Animations
- Room Database History
- Persistent Calculator State
- Light & Dark Theme
- MVVM Architecture
- Material 3 UI

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- DataStore
- Navigation Compose
- Speech Recognizer
